flaON project

This is the continuation of the Avalon project, but more refined and more sophisticated.
Basically, it's a smart home project, conceived & built by me, an under permanent development, functional and active project that runs in my home.
Its core is based on python threads that run on two raspberry pi. That core exchanges data with android terminals, sensors and actuators, and the whole system gives me more confort and modernity.
