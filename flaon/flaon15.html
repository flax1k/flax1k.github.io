flaON15 - the autumn 2021 version

Some new updates for the flaON15, the system core and the app included, are up & running. The latest updates combine the separated MQTT service, with Node Red and Webthing gateway.

The Node Red works very well without any problems from the beginning.
Since the spring's updates, the flaON has a new raspberry pi, that's running a webthing gateway accessible from remote locations. This gateway allows me to check the statuses of various webthings.

The second major change is the background running mqtt service for the app, a stable service that has eliminated the lost connection issues.
Also, a new interface is available for this 15th version (see below):

A clean, simple, fast, fragment based interface. The app has a configurable built-in menu to meet my needs.

I've dropped the full of details view in exchange for a simpler and cleaner look.
