New updates for flaON (version 4.01)

New updates and improvements for my flaON project.
These are the bigest changes made to the core of the flaON system, changes that affect the python scripts and changes to the android terminals.

Python core

The most important change to the python core consists in dividing the processes into classes, running the processes separately in parallel by multiprocessing and implementing the error control through the try - except standard
