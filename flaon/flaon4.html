flaON - versiune a 4-a

A few new changes to my flaON system.

The daydream screensaver has two brightness modes: the day mode - during the daytime, a much brighter value of brightness that makes the display readable and the night mode - during the nighttime with a not disturbing, reduced brightness.

These modes can be set from the Settings section, where a starting time and an ending time of brightness modes can be choose.

Those values will be send and stored to the daydream process that runs on "Barney" raspberry pi.

A minor change but useful is the added text to the notification sound icon displayed on the main screen.
