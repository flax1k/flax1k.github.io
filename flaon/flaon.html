First updates for this year

I changed the main layout. Again. Why? Simple. I need to know from a single look at my tablet, what is the status of the whole system.

I mean, I needed to know what are status of devices, weather, network, events, notifications, in one single look.

After that, if I need more details about something, to be accesible in a different panel/screen.

For example, in the main screen it is shown the current room temperature, but in the temperatures screen it is displayed the temperatures of all rooms of the house and status about the heating devices.

The main screen shows the current outside temperature and weather status, but the weather screen shows detailed info about the current day and a three days weather prediction.
