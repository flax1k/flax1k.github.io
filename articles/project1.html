This is a functional and under continuous development project of a smart home system built with Raspberry Pi, Arduino and Android tablets. The whole system runs in my home.

This is a stand alone project, meaning NO INTERNET CONNECTION is required (like blynk , azure, cayenne, ubidots and other Internet based projects) for the system to run properly. The Internet connection is used for weather status and predictions, but is also used for sending home status to my phone.

Hardware

  • Raspberry Pi 2 Model B - 2pcs
  • Android tablets - 3pcs - wall mounted in different rooms
  • Wemos ESP8266 - 4pcs
  • DHT22 temperature sensors - 4pcs
  • Alexa Echo - 1pc

Software

  • flaON software installed on each android device - coded in java for android KitKat
  • central scripts that run on both RPIs - coded in python for communication, calendar, weather prediction and data management for the whole system

The FLAON system is based on MQTT communication wich has been proved as a more stable, customizable and faster system than TCP client-server/java-python protocol that I used before.

Features

  1. This project uses Openhab2 wich allows me to control/see different parameters from remote android app installed on my phone;
  2. Custom voice notifications uses TTS engine available in English only and notifies me every message/alerts from the system;
  3. JSON data in MQTT protocol for communications between home devices;

The android tablets run their own daydream services featuring a few info like date, time, temperature and actual weather condition.
