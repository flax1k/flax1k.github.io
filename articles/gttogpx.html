Export from Huawei Watch GT to GPX file


The following script will convert any HiTrack file from Huawei watch GT into a STARAVA readable GPX file.

Before converting, it's necessary to ensure to have all needed packages by running the command: pip3 install -r requirements.txt.


where the requirements are: 

numpy==1.16.3
pandas==0.24.2
pkg-resources==0.0.0
python-dateutil==2.8.0
pytz==2019.1
six==1.12.0
